## Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser
